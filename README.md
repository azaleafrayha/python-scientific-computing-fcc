# Scientific Computing with Python

A minimalist collection of Python exercises, mini-projects, and algorithm implementations completed during my learning journey through the freeCodeCamp Scientific Computing with Python curriculum. 

Inside, you'll find everything from basic text manipulation utilities to classic computer science algorithms.

## Projects

* **[Arithmetic Formatter](./arithmetic%20formatter)** — Formats arithmetic problems vertically and side-by-side for better readability.
* **[Bisection Method](./bisection%20method)** — Approximates the roots of a continuous function using numerical techniques.
* **[Convert Text to snake_case](./convert%20text%20to%20snake_case)** — Automatically converts camelCase and PascalCase strings into standard Python snake_case.
* **[Dijkstra Algorithm](./dijkstra%20algorithm)** — Finds the shortest path between nodes in a weighted network graph.
* **[Expense Tracker](./expense%20tracker)** **(Pending)** — A lightweight CLI utility to log, categorize, and track daily expenditures.
* **[Hanoi Recursive Algorithm](./hanoi%20recursive%20algorithm)** — Solves the classic mathematical Tower of Hanoi puzzle using recursion.
* **[Merge Sort Algorithm](./merge%20sort%20algorithm)** — Sorts collections efficiently using a divide-and-conquer sorting method.
* **[Password Generator](./password%20generator)** — Generates secure, randomized passwords based on custom rules and constraints.
* **[Text Encryptor](./text%20encryptor)** — Encrypts and decrypts text messages using basic cryptographic shifting.
* **[Time Calculator](./time%20calculator)** **(WIP)** — Adds a specified duration to a start time, handling AM/PM shifts and days later.
* **[Verifying Card Number](./verifying%20card%20number)** — Validates the mathematical integrity of card numbers using the Luhn algorithm.
